🌍 Travel Website

Um site de viagens moderno e responsivo, desenvolvido para explorar destinos incríveis ao redor do mundo com uma experiência visual envolvente ✈️🌴

✨ Sobre o projeto

Este projeto apresenta um layout elegante e interativo, com foco em destacar destinos turísticos, proporcionando uma navegação simples e agradável para o usuário.

🚀 Tecnologias utilizadas
HTML5 — Estrutura da aplicação
CSS3 — Estilização e responsividade
JavaScript — Interatividade e funcionalidades
Swiper.js — Carrosséis e slides dinâmicos
🎯 Funcionalidades
🌎 Exibição de destinos turísticos
🎞️ Carrossel interativo com Swiper
📱 Design totalmente responsivo
💬 Seção de depoimentos
🎨 Interface moderna e intuitiva
